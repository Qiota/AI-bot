# Discord-бот с ИИ

Этот проект представляет собой Discord-бота, использующего искусственный интеллект для генерации ответов на сообщения пользователей. Бот интегрирован с API Hugging Face, поддерживает контекстную память и предоставляет команды для взаимодействия с пользователями. Также реализован простой веб-сервер Flask для проверки активности бота.

## Основные возможности

- **Ответы на сообщения**: Бот реагирует на упоминания, ответы на свои сообщения или сообщения в личных чатах, генерируя ответы с помощью модели ИИ.
- **Контекстная память**: Сохраняет до 10 последних сообщений в чате для поддержания контекста.
- **Команды**:
  - `/info` — отображает информацию о боте (имя, модель, пинг и т.д.).
  - `/say` — позволяет администраторам или модераторам отправлять сообщения от имени бота.
- **Обновление статуса**: Показывает пинг бота в реальном времени.
- **Веб-сервер**: Flask-сервер на порту 8000 для проверки статуса бота.

## Используемые технологии

- **Python 3.8+**
- **discord.py** — библиотека для работы с Discord API.
- **huggingface_hub** — интеграция с моделями ИИ через Hugging Face.
- **Flask** — легкий веб-сервер.
- **python-dotenv** — загрузка переменных окружения из файла `.env`.

## Установка

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/Qiota/AI-discord.git
   cd AI-discord
   ```

2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

   Содержимое `requirements.txt`:
   ```
   discord.py
   huggingface_hub
   python-dotenv
   flask
   ```

3. **Настройте переменные окружения**:
   Создайте файл `.env` в корневой директории проекта и добавьте следующие переменные:
   ```
   DISCORD_TOKEN=ваш-дискорд-токен
   HF_API_TOKEN=ваш-токен-hugging-face
   PROVIDER=novita
   MODEL_NAME=deepseek-ai/DeepSeek-V3-0324
   MAX_TOKENS=2000
   TEMPERATURE=0.7
   DEVELOPER_ID=ваш-discord-id
   ```
   - `DISCORD_TOKEN` — токен вашего бота (получите в [Discord Developer Portal](https://discord.com/developers/applications)).
   - `HF_API_TOKEN` — токен API Hugging Face.
   - `DEVELOPER_ID` — ваш Discord ID (опционально, для дополнительных прав).

4. **Запустите бота**:
   ```bash
   python bot.py
   ```

## Использование

- Упомяните бота в сообщении (например, `@BotName текст`), ответьте на его сообщение или пишите ему в личку — он сгенерирует ответ.
- Используйте команду `/info` для получения информации о боте.
- Администраторы и модераторы могут использовать `/say текст` для отправки сообщений от имени бота.

## Логирование

Бот использует систему логирования, выводящую информацию в консоль в формате:
```
2025-04-01 12:00:00 [INFO] BotName онлайн!
```

## Примечания

- Бот требует постоянного подключения к интернету для работы с API Hugging Face.
- Команда `/say` доступна только пользователям с правами администратора или модератора (управление каналами/сообщениями).
- Веб-сервер доступен по адресу `http://localhost:8000` и возвращает сообщение "Бот активен!".
